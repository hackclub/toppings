what's this about?

So you've had a look at HTML and JavaScript, and you're getting pretty good at making websites (that might look a little like this)! But now, it's time to add a little more... flavour to them (and make something using these elements.) We've got the perfect opportunity for you to do so.

what will i learn?

  • How to use grids to make fun layouts
  • How to use flexboxes
  • How to use variables in CSS
  • How to make layouts that are responsive to screen size
  • How to animate in CSS
  • How to use margins and padding
  • How to create a gradient background

requirements:

  • Gotta be unique! Have fun with it :)
  • Gotta have one thing animated
  • Gotta have one thing that changes when the screen size is reduced
  • Gotta use containers
  • Gotta either use a grid OR have a gradient background

sooo what do i get?

If your website fulfils all the requirements, you get money to buy toppings for your boba or ice cream!
